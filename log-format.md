# personal-log-hub ログ整形フォーマット
# 更新：2026-04-19

## 使用可能タグ一覧

### 開発
実装・設計・バグ・調査・公開・環境構築・仕様・リファクタ

### 状態・感情
達成・停滞・疲労・集中・迷い・気づき

### その他
日常・体調・筋トレ・読書・振り返り

※該当タグがない場合はClaudeが追加提案する。承認後にこのリストに追記する。

## ログフォーマット

- date: YYYY-MM-DD
- title: （一言タイトル）
- tag: （上記一覧から1つ以上・カンマ区切り）
- mood: （そのときの感情を自由に・任意）
- duration: （分・任意）
- status: （完了／中断／継続・任意）
- content: |
    （内容）

## Claudeへの指示文

作業終了時または会話の区切りで以下を実行する。

1. この会話・作業内容を読んでログを整形する
2. 上記フォーマットに沿って出力する
3. tagは使用可能タグ一覧から選ぶ。該当なければ提案する
4. 複数のログにまたがる場合は分けて出力する
5. 出力はコードブロックで囲む（アプリへの貼り付け用）
