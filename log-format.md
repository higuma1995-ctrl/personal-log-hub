## ジャンル一覧

### 開発・学習

- sheet: YYYY-dev
- tags: 実装・設計・バグ・調査・公開・環境構築・仕様・リファクタ・達成・停滞・疲労・集中・迷い・気づき

```text
- date: YYYY-MM-DD
- title: （一言タイトル）
- tag: （上記一覧から1つ以上・カンマ区切り）
- mood: （そのときの感情を自由に・任意）
- duration: （分・任意）
- status: （完了／中断／継続・任意）
- content: |
    （内容）
```

### 筋トレ

- sheet: YYYY-fitness
- tags: 胸・背中・肩・腕・脚・体幹・有酸素・全身
- 重量・回数・セット・時間は任意
- 有酸素は時間のみ、重量系は重量・回数・セットのみ使う

```text
- date: YYYY-MM-DD
- title: （一言タイトル）
- tag: （上記一覧から1つ以上・カンマ区切り）
- mood: （そのときの感情を自由に・任意）
- 部位: （胸／背中／肩／腕／脚／体幹／有酸素／全身・任意）
- 重量: （kg・任意）
- 回数: （回・任意）
- セット: （セット数・任意）
- 時間: （分・任意）
- content: |
    （内容）
```

## Claudeへの指示文

作業終了時または会話の区切りで以下を実行する。

1. この会話・作業内容を読んでジャンルを選ぶ
2. 上記フォーマットに沿って出力する
3. tagは該当ジャンルの使用可能タグ一覧から選ぶ。該当なければ提案する
4. 複数のログにまたがる場合は分けて出力する
5. 出力はコードブロックで囲む（アプリへの貼り付け用）
